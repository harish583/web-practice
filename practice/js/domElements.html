DOM

  1. The DOM defines a standard for accessing documents
  2. The HTML DOM is a standard for how to get, change, add, or delete HTML elements.
  3. HTML DOM methods are actions you can perform (on HTML Elements).
  4. HTML DOM properties are values (of HTML Elements) that you can set or change.
  5. In the DOM, all HTML elements are defined as objects.
  6. The programming interface is the properties and methods of each object.
  7. A property is a value that you can get or set (like changing the content of an HTML element).
  8. A method is an action you can do (like add or deleting an HTML element).
  9. The HTML DOM document object is the owner of all other objects in your web page.
